# RAG Chatbot Challenge

A Retrieval-Augmented Generation (RAG) chatbot that answers questions from a large private corpus of PDFs, using free/open-source embedding models and vector database.

## Status
🚧 Work in progress

## Tech Stack
- **PDF Extraction:** PyMuPDF, Tesseract OCR
- **Embeddings:** BAAI/bge-small-en-v1.5 (via sentence-transformers)
- **Vector DB:** ChromaDB
- **LLM:** Ollama (local, free)
- **UI:** Streamlit
